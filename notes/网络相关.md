- git 设置与取消代理

```sh
#  设置代理
git config --global http.proxy 'socks4://localhost:1080'
git config --global https.proxy 'socks4://localhost:1080'

# 取消代理
git config --global --unset http.proxy
git config --global --unset https.proxy
```
